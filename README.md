# pappasbrent.github.io

My personal website.

Contains a mix of unique HTML files and HTML files generated by my own custom scripts in the `scripts/` directory which read TSV files in the `tsv/` directory and export them to both HTML and `.typ` files so that I can update my website and CV from a single source.

Structure

```
.
├── 404.html            # Unique HTML file
├── advising.html       # Generated by export scripts
├── assets              # Media files. Typst-compiled CV is placed here
├── blog.html           # Jekyll template file
├── CNAME               # CNAME file for using custom domain name
├── _config.yml         # Jekyll configuration file
├── _drafts             # Jekyll directory for blog posts
├── Gemfile             # Ruby dependencies
├── Gemfile.lock        # Ruby dependencies
├── ignore              # Miscellaneous
├── _includes           # HTML partial templates
├── index.html          # Unique HTML file
├── _layouts            # HTML template files
├── mahjong.md          # Unique HTML file
├── main.css            # Style file
├── Makefile            # Build system
├── news.md             # Unique HTML file
├── _posts              # Jekyll directory for storing posts
├── publications.html   # Generated by export scripts
├── reading.html        # Unique HTML file
├── README.md           # This file
├── research            # Unique HTML files
├── resources.md        # Unique HTML file
├── Roboto-Regular.ttf  # Font file
├── scripts             # Scripts for exporting TSV files to CV and web
├── service.html        # Unique HTML file
├── _site               # Generated by export scripts
├── teaching.html       # Generated by export scripts
├── tsv                 # Data files that export scripts read
└── typ                 # CV template and typ files generated by export scripts
```
